Spotify para Linux

Aqui você pode encontrar diferentes maneiras de instalar o Spotify para Linux. Spotify para a plataforma Linux é um trabalho de amor de nossos engenheiros que queriam ouvir Spotify em suas máquinas de desenvolvimento Linux. E Eles trabalham nisso em seu tempo livre e atualmente não é uma plataforma que apoiamos ativamente. A experiência pode ser diferente de nossos outros clientes Spotify Desktop, como Windows e Mac. Você pode nos dizer o que pensa e pedir ajuda a outros usuários no fórum Desktop (Linux) no fórum da Comunidade Spotify.

Instalação no Ubuntu

Vá para Spotify no Ubuntu Software e clique em instalar. Se o link não funcionar, abra o Ubuntu Software e procure por Spotify.

Install via command line

Debian / Ubuntu

O Spotify para Linux também é lançado como um pacote Debian. Nosso objetivo é que ele funcione com a versão mais recente do Suporte de Longo Prazo do Ubuntu, mas tentaremos fazê-lo funcionar para outras versões do Ubuntu e do Debian também.

Você primeiro precisará configurar nosso repositório debian:

curl -sS https://download.spotify.com/debian/pubkey_5E3C45D7B312C643.gpg | sudo apt-key add - echo "deb http://repository.spotify.com stable non-free" | sudo tee /etc/apt/sources.list.d/spotify.list

Então você pode instalar o cliente Spotify:

sudo apt-get update && sudo apt-get install spotify-client
